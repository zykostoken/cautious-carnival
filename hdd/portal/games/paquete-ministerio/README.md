# Cortadora de Cesped - Juego Terapeutico

## Descripcion del Proyecto

**Cortadora de Cesped** es un juego interactivo desarrollado como herramienta terapeutica para el Hospital de Dia Jose Ingenieros. Esta disenado para asistir en procesos de rehabilitacion cognitiva, estimulacion mental y coordinacion motriz.

## Proposito Terapeutico

El juego esta especificamente disenado para:

- **Estimulacion cognitiva**: Requiere planificacion y toma de decisiones
- **Coordinacion motora**: Control preciso del movimiento (teclado, mouse o tactil)
- **Atencion sostenida**: Mantener el foco durante las sesiones de juego
- **Resolucion de problemas**: Navegar obstaculos y gestionar recursos limitados
- **Progresion gradual**: 60+ niveles con dificultad incremental

## Poblacion Objetivo

- Pacientes en rehabilitacion cognitiva
- Adultos mayores en estimulacion mental
- Pacientes con necesidades de coordinacion motriz
- Usuarios del Hospital de Dia

## Caracteristicas Principales

### Jugabilidad
- Control mediante teclado (flechas), mouse o pantalla tactil
- Sistema de puntuacion motivacional
- Multiples herramientas: cortadora, bolsa de cesped, pala
- Obstaculos variados: piedras, arbustos, agua

### Accesibilidad
- Interfaz clara y de alto contraste
- Textos legibles
- Controles intuitivos
- Funciona en cualquier dispositivo con navegador moderno

### Tecnologia
- Aplicacion web HTML5 autocontenida
- No requiere instalacion
- Sin necesidad de conexion a internet (una vez cargado)
- Compatible con Windows, Mac, Linux, tablets y smartphones

## Requisitos del Sistema

| Componente | Requisito Minimo |
|------------|------------------|
| Navegador | Chrome 60+, Firefox 55+, Safari 11+, Edge 79+ |
| Pantalla | 800x600 pixeles minimo |
| Dispositivo | PC, tablet o smartphone |
| Internet | Solo para carga inicial |

## Como Usar

1. Abrir el archivo `cortadora-de-cesped.html` en un navegador web
2. El juego se carga automaticamente
3. Seguir las instrucciones en pantalla
4. Usar las flechas del teclado o hacer clic/tocar para mover
